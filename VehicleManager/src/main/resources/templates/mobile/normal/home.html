Sistema de Manutenção



Com este sistema é possivel gerir a manutenção de veículos e alfaias bem como manter um histórico do que foi feito a cada uma delas.

Para além destas funcionalidades é possível manter alertas sobre tarefas que devem ser feitas num veículo. Quando for data de realizar a tarefa o sistema envia-lhe um email a avisar.
